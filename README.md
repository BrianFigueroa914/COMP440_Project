# COMP440_Project# COMP440_Project

A sample Electron-based application developed for the COMP440 course. This repository contains the source code for a login interface, dashboard, and utility components.

## 📁 Project Structure

```
COMP440_Project/
  electron/
    main.js
    package.json
    styles.css
  src/
    assets/
    components/
    pages/
      dashboard/
      loginPage/
        index.html
    utils/
```

## 🚀 Getting Started

These instructions will help you run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- npm (included with Node.js)

### Installation

```bash
cd COMP440_Project/electron
npm install
```

### Running the Application

```bash
npm start
```

This should launch the Electron application with the login page.

## 🛠️ Development

- Modify UI in `src/pages` and `electron/main.js` for app logic.
- Styles are located in `electron/styles.css`.
